# Бомжара CEO — заметки разработки

## Текущий статус

Игра: Telegram Mini App кликер  
Репозиторий: bomzhara-ceo  
Главный файл: index.html  
Хостинг: GitHub Pages  
Бот: @bomzhara_ceo_bot

## Уже сделано

- Базовый кликер
- Деньги за клик
- Апгрейд клика: “Рваный стаканчик”
- Пассивный доход: “Шаурма-точка”
- Мем-буст x2 на 30 секунд
- Сохранение прогресса в localStorage
- Подключение Telegram Web App SDK
- Имя игрока из Telegram
- Вибрация при клике
- Ежедневный бонус
- Кнопка “Позвать друга”
- Уровни/ранги
- Прогресс-бар до следующего ранга
- Ранги считаются по totalEarned, а не по текущему балансу money
- Настроен Git + GitHub + GitHub Pages
- Добавлен .gitignore, чтобы не заливать bot-token.txt

## Важно

Токен бота нельзя отправлять на GitHub.  
Файлы bot-token.txt и .env должны быть в .gitignore.

Ранг считается по totalEarned, чтобы игрок не терял статус после покупки улучшений.

## Последнее изменение

Добавлена система заданий в `index.html`:

- Сделать 50 кликов
- Купить 3 улучшения клика
- Купить 2 шаурма-точки
- Забрать ежедневный бонус
- Позвать друга

Награды за задания начисляются в `money` и `totalEarned`, чтобы прогресс ранга тоже рос.
